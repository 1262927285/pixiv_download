# pixiv下载

p站爬虫，以画师pid为参数，一次性下载所有作品(默认为**原图**)  

(**部分**图片需要登录(cookie)才能下载)

部署
--

**linux** 要先装一下依赖:

`cd pixiv_download`

`pip3 install -r requirements.txt`

运行:

`python3 pixiv_download.py`  

**windows** 可以去release里用打包好的版本
